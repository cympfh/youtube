# YouTube playlist

## play

```bash
./play ./lists/double-fav
```

This opens your browser.

## bin/

### getlist

`https://www.youtube.com/playlist?list=$ID` -> list of ids

```bash
./bin/getlist PLV1VKKSui4LHFHClT8VZHpRe_fRJ5iGoI
```

## getinfo

get info a video

```bash
./bin/getlist PLV1VKKSui4LHFHClT8VZHpRe_fRJ5iGoI | ./bin/getinfo
```
