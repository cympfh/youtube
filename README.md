# YouTube playlist

## play playlist files

```bash
./play -o --shuffle playlist1 playlist2
```

## make your playlist

```bash
# from playlist url
./get https://www.youtube.com/playlist?list=PLV1VKKSui4LHL2aiQV-K_CLg6bwj45u-F >> playlist

# from a music page url
./get https://www.youtube.com/watch?v=kZmk8PxjhnY >> playlist
```
